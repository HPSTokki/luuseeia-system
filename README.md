## Setup

Make sure to install dependencies:

```bash
# npm
npm install

# bun
bun install
```

## Development Server

Start the development server on `http://localhost:3000`:

```bash
# npm
npm run dev

# bun
bun run dev
```

## Production

Build the application for production:

```bash
# npm
npm run build

# bun
bun run build
```

Locally preview production build:

```bash
# npm
npm run preview

# bun
bun run preview
```
